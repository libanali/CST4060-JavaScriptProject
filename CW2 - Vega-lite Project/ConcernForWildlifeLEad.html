Concern for wildlife - visual effectiveness

In this section I will be discussing the visual mapping of my finding and why I chose them. I will be discussing the marks, channels and attributes.

As you can see in my visualisation for Lead, I used a line chart to help me discover if there were any concerns for wildlife in the present or future. As mentioned before, a line chart is used to display changes over time as a series of data points connected by a straight line. The chosen chart type for this the best fit for this finding because line charts were specifically designed to help users discover changes over time. This was very convenient for me because one of the attributes that were used for the visualisation was ‘sample date’ which gives information on the days, months and years. More than one mark was used in this visualisation and they are point (0D) and line (1D). The line was used to help the user track any changes over time and the point was used to help the user identify specific values on the graph. The visualisation falls under the identity channel because of the way I identified the chemical by using colour hues. Colour hues are considered to be one of the best channels for categorical data as stated by Munzner in “Visualisation analysis and design” (Munzner, 2014). It states, “The most effective channel for categorical data is spatial region, with colour hue as the next best one” (Munzner, 2014). Although it is the second best, I thought a set of colours would be best way to highlight the story that I want the data to tell. There is one additional feature that was applied when designing these graphs and that was tooltip. Tooltip allows the user to display the values by simply hovering their mouse over the graph.
